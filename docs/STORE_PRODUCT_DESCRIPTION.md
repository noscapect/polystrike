# Polystrike Arena - Store Product Description

## Short description

Fast-paced Instagib arena shooter with in-app map editing, shareable full-map files, and bots that learn each level over time.

## Full description

**Polystrike Arena** is a classic arena FPS experience built for players who love speed, precision, and replayability.

Jump into high-intensity Instagib combat, then go beyond playing: edit maps directly in-game, share full map files with other players, and train bots that continuously learn how to navigate each arena.

Whether you want quick frag-heavy matches, CTF rounds, custom map experimentation, or AI behavior tuning, Polystrike Arena gives you a complete player-and-creator loop in one package.

## Key features

- **Classic arena gameplay** with both Instagib and Classic DM rulesets
- **Singleplayer + CTF support** with adjustable match settings
- **In-game map editor** (place/remove blocks, size/material/rotation controls)
- **Built-in map override system** so edited maps can be played in normal singleplayer
- **Local map publishing** with up to **99** personal map slots
- **File-based map sharing** via **full-map JSON export/import**
- **Adaptive bot navigation learning** with optional always-learn behavior
- **Bot difficulty tiers (1-5)** from *Fresh Meat* to *Unreal*
- **Debug overlays** for nav graph and combat heatmap hotspot analysis

## Why players like it

- Easy to start, hard to master
- Strong replayability through custom maps and adaptive bots
- Great for both casual matches and technical experimentation
- No external editor required to create map variants

## Modes and content

- **Deathmatch Instagib**
- **Capture The Flag**

## Controls at a glance

- Move: WASD  
- Look: Mouse  
- Fire: Left Mouse  
- Jump: Space (double jump enabled)  
- Dodge: double-tap WASD
